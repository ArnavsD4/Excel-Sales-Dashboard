# 📊 Excel Sales Dashboard

## 📌 Overview
This project presents an interactive Excel dashboard built to analyze sales data and generate business insights.

## 🚀 Features
- Data cleaning and preprocessing
- Pivot tables for analysis
- Charts (Bar, Pie, Line)
- Monthly sales trend analysis
- Interactive slicers (Region, Category)

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## 📈 Insights
- Identified top-performing regions
- Compared category-wise sales distribution
- Analyzed monthly sales trends

## 📂 Files
- sales_dashboard.xlsx → Final dashboard
- train.csv → Dataset
